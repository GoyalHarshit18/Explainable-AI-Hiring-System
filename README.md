# 📌 Overview

Traditional Applicant Tracking Systems (ATS) rely heavily on **keyword matching**, often overlooking qualified candidates and providing no transparency behind ranking decisions.

This project introduces an **Explainable AI Hiring System** that combines:

- 🔍 Hybrid Candidate Retrieval
- 🧠 Semantic Understanding
- 📈 Career Evidence Modeling
- ⚡ Intelligent Feature Engineering
- 💬 Dynamic Recruiter Reasoning

to produce **accurate, explainable, and recruiter-friendly candidate rankings**.
